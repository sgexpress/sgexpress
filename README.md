# 简介

## 关于

《三高特快车》是武汉市第三中学的非正式电子刊物，与武汉市第三中学校方无关，为武汉市第三中学学生出于兴趣建立，旨在在文学社刊物《汉阳树》、《武汉三中》QQ频道、校园墙之外，为同学们提供另一个广阔的天地。
