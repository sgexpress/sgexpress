# 简介

## 关于

《三高特快车》是武汉市第三中学的**非正式电子刊物**，**与武汉市第三中学校方无关**，为武汉市第三中学学生**出于兴趣**建立，旨在在文学社刊物《汉阳树》、《武汉三中》QQ频道、校园墙之外，为同学们提供另一个**广阔的天地**。

我们的**原则**是：可以不**高雅**，可以不**深刻**，但是要**自然**。
